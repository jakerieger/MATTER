Detailed Description

This is the camera used in the scene view. It implements the Arcball (aka Orbit) camera mode. FOV, aspect ratio, and near/far planes are configurable but the defaults are recommended.

This is a header-only class with all the implementation done here.

Remarks
I based my implementation on the wonderful tutorial by Marie Eve Dube here: https://asliceofrendering.com/camera/2019/11/30/ArcballCamera/
Authors
Jake Rieger, Marie Eve Dube
